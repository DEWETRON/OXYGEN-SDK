=============================
Example: Video Source
=============================

This example demonstrates how to create a video source channel that creates new images at a specified sample rate

---------
Features
---------
  * Register a new Software Channel Type in Oxygen
  * Plugin instance provides scalar config items for image size, frame rate and color format
  * Provides internal config items to support compressed video storage in Oxygen
  * Generate a test video image with a configurable sample rate
